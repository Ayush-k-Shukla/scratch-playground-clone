# Scratch playground

To get started:

- Run `npm i` to install dependencies
- Run `npm start` and open http://localhost:3000 to see the app

Need to implement

1. Motion
   a. move `x` steps
   b. turn `x` degree clockwise
   c. turn `x` degree anti-clockwise
   d. go to `x` and `y`
2. Looks
3. Control
4. Events
5. Able to create multiple sprites
